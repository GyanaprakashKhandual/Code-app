## 📖 About This Portfolio

Welcome to my interactive portfolio! This project is built using **Next.js**, styled with **Tailwind CSS**, and powered by **Framer Motion animations** to create a **professional, modern frontend experience**.

> 💡 This isn't just a portfolio – it's a reflection of my journey, my skills, my passion, and everything I've learned so far in **web development**, **automation testing**, **ethical hacking**, and more.

---

## ✅ Features Showcase

- ✔️ **About Me** — My background, passion, and personal story.
- ✔️ **Skills & Technologies** — All tech stacks I'm familiar with.
- ✔️ **Projects** — Featured real-world projects with links.
- ✔️ **Education** — Academic journey with certifications.
- ✔️ **Client & Peer Feedback** — What others say about my work.
- ✔️ **Blog Posts** — Articles I've written on testing, security, and dev topics.
- ✔️ **Modes & Commands** — Terminal commands I frequently use.
- ✔️ **Dark/Light Modes** — User toggle for visual preference.
- ✔️ **Professional Design** — Interactive animations, icons, and effects.
